# 🛒 Smart E-Commerce Customer Intelligence System (ML + Streamlit)====(smartshop AI)

## 🚀 Project Overview

This project is a simple but realistic **AI-powered e-commerce system** built using Machine Learning and Streamlit.

It simulates how online stores analyze customer purchase behavior to predict customer segments and improve business decisions.

The system allows users to input purchase details (date, products, quantity, price) and automatically predicts their customer segment based on historical behavior.

---

## 🧠 Key Features

- 📊 Customer behavior analysis using real transaction data  
- 🧾 Multi-product invoice simulation (like real e-commerce orders)  
- 🔄 Automatic feature mapping from product selection  
- 🤖 Customer segmentation prediction using Machine Learning model  
- 📦 RFM-based customer profiling (Recency, Frequency, Monetary)  
- 🛍️ Support for multiple products per single invoice  
- ⚡ Interactive UI built with Streamlit  

---

## 🧪 Machine Learning Pipeline

1. Data Cleaning & Preprocessing  
2. Feature Engineering (RFM Analysis)  
3. Customer Segmentation using K-Means  
4. Classification Model for Segment Prediction  
5. Recommendation-ready structure for future expansion  

---

## 💡 Project Idea

The system mimics a real-world e-commerce environment where:

- A user can create a purchase invoice  
- Select multiple products in one transaction  
- The system automatically tracks behavior  
- A trained ML model predicts the customer segment  

This helps businesses understand:
- Who are VIP customers  
- Who are at risk of leaving  
- Who are regular buyers
